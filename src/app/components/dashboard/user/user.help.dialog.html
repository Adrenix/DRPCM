User Component Help

This component displays the current connection to your discord client. In order to successfully apply a
RPC to your discord client, discord must be connected to DRPCM. If a connection cannot be established,
you will be notified to try again. An application client identifier is required to preload assets. Use the help
menu located in the top left for more information about RPC applications.

In the submenu, clearing the identifier removes the ID currently present in the RPC client ID field. You will
be given an opportunity to undo the action if needed.
