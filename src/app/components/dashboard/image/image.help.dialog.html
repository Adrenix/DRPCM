Image Component Help

This component allows you to control what images appear in your custom RPC. In order to obtain a list of assets,
you need to give a RPC client identifier in the field above this component. Once an identifier is provided, use the
Refresh Lists button to receive an updated list of assets available from the RPC application given.

In the submenu, clearing the fields removes selected images and hover texts. You will be given an opportunity
to undo the action if needed.
