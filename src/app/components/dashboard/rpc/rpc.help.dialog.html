RPC Component Help

This component allows you to preview what your RPC will look like before applying it. When you are
satisfied with the result, click the Start button to apply the RPC to your discord client. Use the Stop
button to remove the RPC from your discord client.

In the submenu, saving the RPC will allow DRPCM to remember your RPC after the program has been
closed. Resetting the RPC will clear all fields in each component. You will be given an opportunity
to undo the action if needed.
